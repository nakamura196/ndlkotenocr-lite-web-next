1. 概要

このODD (One Document Does it all) は、NDL古典籍OCR-Liteアプリケーションが生成するTEI XMLドキュメントの最小限のスキーマを定義します。実際に使用する要素と属性のみを含みます。

2. Overview

This ODD (One Document Does it all) defines the minimal schema for TEI XML documents generated by the NDL Koten OCR-Lite application. It includes only the elements and attributes actually used.

3.4. Constraints

Schematron
<sch:rule context="tei:lb[@corresp]"> <sch:let name="zoneId"  value="substring-after(@corresp, '#')"/> <sch:assert test="//tei:zone[@xml:id = $zoneId]"> 行ブレークは存在するゾーンを参照する必要があります / Line break must reference an existing zone </sch:assert> </sch:rule>
Schematron
<sch:rule context="tei:lb[@n]"> <sch:assert test="matches(@n, '^\d+\.\d+$')"> 行番号は次の形式である必要があります: ページ.行 (例: 1.1, 2.3) / Line number must be in format: page.line (e.g., 1.1, 2.3) </sch:assert> </sch:rule>
